 ## Run server & client
  _Run Server_
1. In server : Port 8081
    ```sh
     nodemon src/app.js
    ```
  _Run client_

2. In client : Port 8080
    ```sh
   $ npm run dev
    ```
## Install npm server & client
  ```sh
    $ npm install
   ```
